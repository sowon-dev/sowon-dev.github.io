클릭이벤트 : click(), bind(), on()

  • 참고링크 : click()이벤트
  • bind(“이벤트종류”, 실행코드) : 해당요소에 이벤트를 등록시키는 함수
  • click(function(){}) : 클릭이벤트 함수
  • on(“이벤트종류”, 실행코드) : 이벤트처리함수
  • 체이닝기법 : 점을 통해서 함수를 이어주는 기법이다.

  • unbind(); : 한번만 실행하고 이벤트종료
